# 💬 YallaChat

A real-time chat application built with modern web technologies, allowing users to communicate instantly with a smooth and interactive UI.

---

## ✨ Features

- User authentication (Sign up / Login)
- Real-time messaging
- Online / Offline user status
- Media sharing (images/videos/audios/files)
- Instant message delivery
- Organized chat conversations
- Voice call & Voice record
- Responsive and modern UI

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Supabase (for database & real-time communication)
- Firebase (for notifacations)
- Cloudinary (for media uploads)
- Axios
- Tailwind Css
